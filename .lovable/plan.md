
# AirML — AI in Aerospace Engineering Portfolio Website

## Overview
A clean, professional academic portfolio website that describes AI applications in aerospace engineering and showcases the AirML project with a simulated demo of aircraft engine remaining useful life (RUL) prediction.

---

## Page 1: Landing / Hero
- Bold headline: "AI-Powered Aircraft Predictive Maintenance"
- Brief tagline about using machine learning to improve aviation safety
- Navigation bar with links to all sections
- Hero image/illustration area with aerospace theme

## Page 2: About AI in Aerospace Engineering
- Informational section explaining how AI/ML is used in aerospace
- Key areas covered: predictive maintenance, engine health monitoring, flight safety
- Cards highlighting use cases (turbofan degradation, sensor analysis, RUL prediction)
- Reference to NASA's Turbofan Engine Degradation Simulation Dataset

## Page 3: The AirML Project
- Project overview: goals, methodology, and tech stack
- **Training Pipeline Section**: Explanation of data cleaning, feature engineering, and algorithms used (Random Forest, XGBoost, Neural Network)
- Sample data visualization using Recharts — showing sensor readings, cleaned data patterns
- Results comparison chart showing model performance

## Page 4: Simulated Demo
- Interactive form where users input simulated engine sensor readings (21 sensors + 3 operational settings)
- Pre-filled sample values for easy testing
- "Predict" button that returns a simulated RUL prediction with a visual gauge/progress indicator
- Results displayed with charts showing predicted remaining cycles and engine health status
- Mock confidence scores and model comparison

## Page 5: Architecture / How It Works
- Flowchart-style section showing the data pipeline (raw data → cleaning → feature engineering → model → prediction)
- Frontend/Backend architecture diagram
- Technologies used section

## Page 6: Team & Contact
- Credits to original developers (Adesh Gautam, Kunal Sharma, Shreya Gupta)
- Link to the GitHub repository
- Contact information section
